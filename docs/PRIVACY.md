# Batterygic Privacy Policy

Last updated: May 7, 2026

Batterygic does not collect, sell, share, or transmit personal data.

## Data stored locally

Batterygic stores app settings on your device. These settings may include:

- Theme and style preferences
- Tray icon preferences
- Notification preferences
- Refresh interval
- Known Bluetooth device names, identifiers, display names, and color/icon preferences

This data is stored locally on your Windows user profile and is used only to run the app.

## Network and analytics

Batterygic does not use analytics, advertising, tracking, accounts, telemetry, or cloud sync.

Batterygic does not upload Bluetooth device information or settings to any server.

## Device access

Batterygic reads local Windows Bluetooth/device information so it can display battery status for supported devices.

## Deleting data

You can remove saved device entries inside the app by forgetting devices. You can also delete the app settings file from your Windows user profile or uninstall the app.

## Contact

For privacy questions, open an issue in this support repo.
